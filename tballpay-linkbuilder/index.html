TBALL Pay is a fully client side utility for generating Solana payment links and HTML buttons. No servers, no databases, no external requests. Just static JavaScript running in the browser.

The builder is deterministic and portable. It can be self hosted, or deployed via IPFS, GitHub Pages, or even from local files. Inputs are encoded into URLs or anchors. nothing is stored or transmitted.

An internal SDK supports transaction creation, simulation, and confirmation using the Solana Web3.js library. Wallet adapters are injected at runtime. All RPC calls are made directly from the client to public Solana endpoints.

No telemetry. No opaque backend. Just inspectable code that moves tokens.
